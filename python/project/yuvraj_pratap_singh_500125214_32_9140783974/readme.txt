# College Food Court Bill Generator

This project is a CSV to PDF converter designed specifically for generating bills for the food court in our college.

## Features

- Convert CSV files containing items and prices to PDF bills.
- Automatically align items and prices in the PDF.
- Calculate discounts and include them in the bill.

## Usage

1. Prepare a CSV file with the items and prices.
2. Run the `main.py` script and follow the prompts to enter customer details, items, and discounts.
3. The program will generate a PDF bill with the specified details.



Python 3.x
fpdf library
Installation
Clone this repository to your local machine.
Install Python if you haven't already.
Install the fpdf library by running pip install fpdf.
